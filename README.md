## Project 1: Newton Fractal Generator
For the first project, I decided to make a Newton fractal generator. Initially, the goal was to create a simple but effective fractal generator utilizing only the CPU. This later changed to creating a fractal generator that can utilize a NVIDIA GPU. For this, I am using NVIDIA's Cuda library.
